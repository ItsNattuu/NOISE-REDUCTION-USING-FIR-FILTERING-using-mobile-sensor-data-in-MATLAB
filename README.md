Noise Reduction Using FIR Filtering (MATLAB)
Overview
This project focuses on reducing noise in real-world mobile sensor data using Finite Impulse Response (FIR) filtering in MATLAB. Sensor data such as accelerometer and gyroscope readings often contain noise due to environmental disturbances, motion artifacts, and hardware limitations. This project demonstrates how digital signal processing techniques can be applied to improve signal quality and reliability.

Features
Design and implementation of FIR filters
Noise reduction in mobile sensor data
Visualization of original and filtered signals
Time-domain and frequency-domain analysis
Simple and modular MATLAB code
Technologies Used
MATLAB
Signal Processing Toolbox
Dataset

The project uses mobile sensor data stored in CSV or Excel format. The data may include accelerometer or gyroscope readings collected from a smartphone or wearable device.

Methodology
Import sensor data into MATLAB
Analyze the characteristics of the noisy signal
Design an appropriate FIR filter (such as a low-pass filter)
Apply the filter to the signal
Compare and visualize the original and filtered outputs
